# HWPX training documents

Place 업무 종류별 `.hwpx` reference files in the folders below.
Firebase Functions reads these files at runtime and uses their extracted text
as local examples when generating a new HWPX document.

- `family_notice/`: 가정통신문
- `work_plan/`: 업무추진 계획
- `education_week/`: 교육주간 운영계획
- `learning_material/`: 학생 학습자료 제작
- `counseling_plan/`: 상담 계획서
- `multicultural_support/`: 다문화 학생 지원 계획

Only `.hwpx` files are parsed. Placeholder files are kept so the folder
structure is committed even before real school templates are added.
